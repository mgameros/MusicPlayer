  Music Playing : System
  User : Actor
  USB Drive : Actor
  Play Music : Use Case

The user selects a song and presses the play button. The system plays the song, enables the pause button, and disables the play button.

  Pause Music : Use Case

The user presses the pause button. The system pauses the currently playing music, enables the play button, and disables the pause button. It also makes note of how far along the current song was.

  Stop Music : Use Case

The user presses the stop button. The system stops the music and does not make note of where the song was. The play button is enabled and the pause and stop buttons are disabled.

  Resume Music : Use Case

The user presses the play button to resume the music that was currently playing. The system finds where the song had been playing and resumes playing. The play button is disabled and the pause button is enabled.

  Switch Song : Use Case

The user chooses a different song while another song is playing. The system enables the play button.

  Change Song : Use Case

The user has selected another song and pressed play while another song is playing. The system stops the current song, finds the selected song in the USB drive, and plays it. The play button is disabled.
