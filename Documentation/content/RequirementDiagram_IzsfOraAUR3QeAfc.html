Model Elements
Name Description
  File Playing : Requirement

The AFP shall play files of the WAV format.

  File Storage : Requirement

Audio files shall be stored on a USB drive connected to the STM32F4-Discovery board.

  Output : Requirement

The audio output shall be through the CS43L22 audio DAC.

  User Interface : Requirement

The user interface shall be on a PC connected to the STM32F4-Discovery board. A GUI interface shall be used.

  File Indexing : Requirement

The AFP shall allow indexing through the files on the thumb drive through the user interface.

  File Name Display : Requirement

The AFP shall display the name of the file on the user interface.

  Selected File Playing : Requirement

The AFP shall allow playing the displayed audio file.

  Playback Pausing : Requirement

The AFP shall allow pausing playback.

  Playback Resuming : Requirement

The AFP shall allow resuming playback.

  File Stopping : Requirement

The AFP shall have a stop mode where no file is playing.

  Initial Mode : Requirement

When the AFP starts there shall be no file playing.

  LED Use : Requirement

LEDs can be used in the system to indicate operating modes.
